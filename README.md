# Testable Profile Card

A responsive profile card project built with semantic HTML, modern CSS, and vanilla JavaScript.

## GitHub Repository

https://github.com/mnotalone/Stage-1B-Testable-Profile-Card.git

## Features

- Semantic markup with accessible structure
- Live epoch time display
- Avatar image support for local upload or URL input
- Social links that open in a new tab
- Responsive layout for mobile, tablet, and desktop

## Run Locally

### Option 1: Open directly

1. Clone or download the repository.
2. Open `index.html` in your browser.

### Option 2: Use VS Code Live Server

1. Open the project folder in VS Code.
2. Install the Live Server extension if needed.
3. Right-click `index.html` and choose "Open with Live Server".

## Project Structure

- `index.html` - page markup and content
- `styles.css` - layout and visual design
- `script.js` - interactivity and live time updates
- `Image.png` - default avatar image
- `fonts/` - self-hosted font files

## Notes

- The default avatar is loaded from `Image.png`.
- Social links and avatar sources can be edited from the control panel in the page.
